# workout-cli

## Install

```
$ npm install -g pm2
$ pm2 startup # instruction for setup autostart of pm2 after reboot
$ npm install -g workout-cli # in furure, now just clone repo && npm link
```

## Usage

```
$ workout --start
$ workout --stop
```
